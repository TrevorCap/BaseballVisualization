Welcome to my collection of baseball stats and visualizations! First you will find two interactive maps for the birth and deathplace of every player in major league history and a time-lapse player size chart that help visualize how baseball players have changed in size by birthplace over time.

The hope of this website is to allow the viewer to visually demonstrate both the concentration and the spread of baseball as a global sport over time.

For the curious, this website is built using a Flask app which queries a SQL databse to produce Leaflet maps and a d3 chart. See also a predictive model with a similarly built website at MLBPredictor. Enjoy!
